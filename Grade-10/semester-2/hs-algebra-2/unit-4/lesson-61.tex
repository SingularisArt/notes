\lesson{61}{Mar 18 2022 Fri (17:53:33)}{Find the maximum or minimum of a quadratic function}{Unit 4}

A supply company manufactures copy machines. The unit cost $C$ (the cost in
dollars to make each copy machine) depends on the number of machines made. If
$x$ machines are made, then the unit cost is given by the function
$C(x) = 0.5x^{2} - 280x + 52,506$. Let's try to find the minimum cost.

\begin{note}
  \label{nte:quadratic_notice}

  The equation has a squared variable, which indicates that it's a quadratic
  equation (\ref{rve:quadratic_equation}). Also, the leading coefficient is
  positive, which means the parabola opens up, which makes the vertex a minimum
  and not a maximum.

  \begin{figure}[H]
    \centering
    \incfig{minimum-maximum-of-quadratic}
    \label{fig:minimum_maximum_of_quadratic}
  \end{figure}

  In this case, when you're asked for the minimum of a quadratic, you're being
  asked to find the vertex of the parabola of the quadratic.
\end{note}

\paragraph{Minimum}
\label{par:minimum}

The function reaches its minimum value at the vertex.
So to find the minimum unit cost, we find the $y$-coordinate of the vertex.

The vertex for the function $C(x) = ax^{2} + bx + c$ occurs when
$x = -\frac{b}{2a}$

So, we get the following:

\begin{equation}
  \label{eqt:the_following}
  \begin{split}
    x = -\frac{b}{2a} = \frac{-280}{2(0.5)} = 280
  \end{split}
.\end{equation}

To find the minimum unit cost, we evaluate $C(280)$.

\begin{equation}
  \label{eqt:evaluate_c_of_x}
  \begin{split}
    C(280) = 0.5(280)^{2} - 280(280) + 52,506 = 13,306
  \end{split}
.\end{equation}

So, the minimum cost is $\$13,306$.
